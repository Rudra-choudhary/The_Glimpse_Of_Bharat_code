# The_Glimpse_Of_Bharat_MAIN
The glimpse of Bharat

Brief Description of the Project

This project is about creating a large database of Freedom Fighters of India even those which are not known on the national Scale 

background: i am a history nerd and when i came across a freedom fighter name i want find out more about him but when i googled it or search it on wikipedia i was not able to find sufficient information about him and from there this idea arise  

problem: enough data of our freedom struggle and important personalities is not available easily

solution: a database which will serve as a one stop solution (which will be openly contributed by general public)

Key Features and Functionality


the website will have a search feature to search the freedom fighters
and a feature to upload them
in the search option they would have options to search by region, party, ideology, contribution etc,

to upload the data on the website we are focusing on two methods
1. data from existing sources like websites (https://amritmahotsav.nic.in/unsung-heroes.htm
https://www.mha.gov.in/sites/default/files/UpdatedlistofSSSYBeneficiaries_04022022.pdf
https://www.constitutionofindia.net/the-constitution-framers/
and many more), national archives, libraries and museums 
2. data from general public (which is also unique selling point ): the user will be able to contribute to the website by providing data about unsung freedom fighters from there area/community
the challenge here is filtering the data for which we need to create some filters

financing the project : although this project is not meant to be a profit making project but to make it sustainable we will provide merch 

it will be like wikipedia but focused on India struggle for freedom,

afterwards we can extend it to include details about the parties and social groups involved

and then we can extend it to include more history about India or maybe the world
it can also be converted to a database of freedom fighters from all other countries in the world and become a global product
